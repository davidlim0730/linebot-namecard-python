# 產品功能清單 — 智慧名片管理 LINE Bot

**版本**：Phase 3（已上線）  
**目標用戶**：台灣企業業務團隊  
**核心價值**：用 LINE 就能管名片——無需另裝 App，共享名片庫、快速查找聯絡人

---

## 功能總覽

| 功能模組 | 功能名稱 | 說明 | 入口 | 狀態 |
|---------|---------|------|------|------|
| **名片新增** | 單張上傳 | 傳送名片照片，AI 自動辨識並儲存 | Rich Menu → 新增名片 → 單張上傳 | ✅ 上線 |
| | 批量排程上傳 | 一次傳送多張名片，系統排程處理後推播結果 | Rich Menu → 新增名片 → 批量上傳 | ✅ 上線 |
| **名片查詢** | 智慧搜尋 | 輸入任意關鍵字（姓名、公司、職稱）自動搜尋 | Rich Menu → 智慧搜尋 | ✅ 上線 |
| **名片管理** | 編輯名片 | 修改已儲存名片的任一欄位 | 名片 Flex Message → 編輯 | ✅ 上線 |
| | 新增備忘錄 | 為名片加上個人備註（例如：合作背景、下次跟進） | 名片 Flex Message → 備忘錄 | ✅ 上線 |
| | 刪除名片 | 從共享名片庫中移除名片 | 名片 Flex Message → 刪除（管理員） | ✅ 上線 |
| | 清理重複名片 | 自動偵測並清除 Email 重複的名片 | Rich Menu → 管理名片 → 清理重複名片 | ✅ 上線 |
| | vCard / QR Code | 產生 QR Code，掃描即可將名片存入手機通訊錄 | 名片 Flex Message → QR Code | ✅ 上線 |
| **群組管理** | 群組分類 | 為名片套用群組標籤（如：VIP客戶、供應商） | Rich Menu → 管理名片 → 群組管理 | ✅ 上線 |
| | 依群組篩選 | 點選群組查看該分類下的所有名片 | 群組分類清單 → 點選群組 | ✅ 上線 |
| | 新增 / 刪除群組 | 管理員可自訂群組名稱 | 群組分類清單 → 管理群組（管理員） | ✅ 上線 |
| **資料匯出** | CSV 匯出 | 將完整名片庫匯出為 CSV，寄送至指定 Email | Rich Menu → 資料與設定 → 匯出 CSV | ✅ 上線 |
| | Google Sheets 同步 | 每次新增 / 更新名片時自動同步至指定試算表 | 自動觸發（需設定 `GOOGLE_SHEET_ID`） | ✅ 上線 |
| **統計** | 名片統計 | 查看總名片數、本月新增數、最常合作公司 | Rich Menu → 資料與設定 → 統計 | ✅ 上線 |
| **團隊協作** | 查看團隊資訊 | 顯示團隊名稱、成員人數、個人身份 | Rich Menu → 團隊功能 → 查看團隊資訊 | ✅ 上線 |
| | 查看成員清單 | 列出所有團隊成員（含 LINE 暱稱） | Rich Menu → 團隊功能 → 查看成員 | ✅ 上線 |
| | 邀請成員 | 管理員產生邀請碼，有效期 7 天 | Rich Menu → 團隊功能 → 邀請成員 | ✅ 上線 |
| | 加入團隊 | 輸入邀請碼加入既有團隊共享名片庫 | Rich Menu → 團隊功能 → 加入團隊 | ✅ 上線 |
| | 設定團隊名稱 | 管理員修改團隊顯示名稱 | 文字指令：`設定團隊名稱 <名稱>` | ✅ 上線 |

## Rich Menu 導覽架構

```
Rich Menu（底部常駐）
├── 名片操作
│   ├── ➕ 新增名片
│   │   ├── 📸 單張上傳（傳照片即可）
│   │   └── 🗂️ 批量上傳（多張排程處理）
│   ├── 🔍 智慧搜尋（輸入關鍵字）
│   └── 📋 管理名片
│       ├── 🏷 群組管理
│       └── 🧹 清理重複名片
├── 團隊功能
│   ├── 👥 查看團隊資訊
│   ├── 👤 查看成員
│   ├── 📨 邀請成員（管理員）
│   └── 🔗 加入團隊
└── 資料與設定
    ├── 📊 統計
    └── 📤 匯出 CSV
```

## 技術亮點（GTM 用）

- **零 App 安裝**：以 LINE 為唯一介面，降低導入門檻
- **AI OCR**：Gemini Vision 自動辨識名片文字，準確率高
- **共享名片庫**：以組織為單位共享，支援多成員協作
- **即時同步**：可串接 Google Sheets，隨時開啟試算表查詢
- **安全隔離**：不同組織資料完全隔離，無資料外洩風險
