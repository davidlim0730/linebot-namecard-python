# Design Brief — LINE Rich Menu 主選單圖片

**專案**：名片管理 LINE Bot（企業業務團隊用）  
**用途**：LINE 聊天室底部固定選單，取代文字指令作為主要導覽入口

---

## 規格

| 項目 | 數值 |
|------|------|
| 尺寸 | **2500 × 843 px** |
| 格式 | PNG |
| 色彩模式 | RGB |
| 檔名 | `rich_menu_main.png` |

---

## 版型：3 欄均分

```
┌─────────────────┬─────────────────┬─────────────────┐
│                 │                 │                 │
│    名片操作      │    團隊功能      │   資料與設定     │
│   (0–833px)     │  (833–1666px)   │ (1666–2500px)   │
│                 │                 │                 │
└─────────────────┴─────────────────┴─────────────────┘
              全寬 2500px，高 843px
```

每欄各寬 833 / 833 / 834px，高 843px，三欄**點擊區域獨立**，無需視覺分隔線（但可以有）。

---

## 各欄內容

| 欄位 | 標題 | 副文字建議 | 圖示方向 |
|------|------|-----------|---------|
| 左 | **名片操作** | 新增・搜尋・管理 | 名片 / 掃描 |
| 中 | **團隊功能** | 成員・邀請・資訊 | 人群 / 團隊 |
| 右 | **資料與設定** | 匯出・Sheets 同步 | 圖表 / 齒輪 |

---

## 風格指引

- **受眾**：台灣企業業務人員，日常工作情境使用
- **調性**：專業、簡潔，略帶科技感；避免過度花俏
- **顏色建議**：可參考 LINE 品牌綠（`#06C755`）做 accent，底色深色或淺色均可，但文字需高對比確保可讀性
- **字型**：中文粗體（e.g. Noto Sans TC Bold），大小視覺層次清楚即可
- **圖示**：建議使用線條型（outline）icon，每欄一個主圖示搭配標題

---

## 技術限制

- 圖片為**靜態圖**，不支援動畫
- LINE 會在圖片上疊加點擊區域，設計本身不需要加按鈕效果
- 三欄之間若有分隔線，建議用細線或留白，避免與點擊邊界錯位
- 行動裝置上顯示寬度約 375px，圖示與文字不宜過細
