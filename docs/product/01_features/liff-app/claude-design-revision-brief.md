## 🔴 必改（高優先）

### 1. 介面文字全面繁體中文化

現有 Claude Design 初稿中所有英文 UI 字串須改為繁體中文：

| 現有英文 | 改為 |
|---------|------|
| Card Manager | 名片管理員 |
| Edit Card Details | 編輯名片 |
| Update Card | 更新名片 |
| Cards | 名片 |
| Team | 團隊 |
| Settings | 設定 |

**範圍**：頂部 Bar 標題、底部 Navigation Bar 標籤、按鈕文字、所有 UI label。  
**不需改**：Email 地址、URL、使用者輸入的資料內容。

## 🟡 應改（中優先）

### 3. 名片總表補上筆數提示

**位置**：篩選 Tab 列下方

**三種狀態都需要設計**：
```
預設：「共 N 張名片」
搜尋中：「找到 N 筆」
搜尋無結果：「找不到符合『○○』的名片」
```

規格：12px，顏色 #3C4A3C，padding-top 8px，左對齊。

---

### 4. Toast 位置統一

**問題**：儲存失敗的錯誤提示目前位置與儲存成功 Toast 不一致。

**統一規格**（兩者相同位置）：
```
位置：頂部 Bar 下方，全寬，貼邊（圓角 0px）
高度：44px
成功：#06C755 底，白字
失敗：#C0000A 底，白字
動畫：slide-down 200ms 出現；fade-out 300ms 消失
```

**需要修改**：將儲存失敗 Toast 從現有位置移至頂部 Bar 下方。

---

### 5. 名片詳情補上備忘錄快捷操作

**位置**：名片詳情頁 → 備忘錄區塊

**現有**：備忘錄區塊只有顯示文字，無備忘錄時顯示「尚未新增備忘錄」。

**新增**：
```
無備忘錄時：
「尚未新增備忘錄」 + 「＋ 快速新增」連結（#006E2B 文字）
點擊 →  同一頁 inline 展開 textarea（不跳至完整編輯頁）

有備忘錄時：
備忘錄文字 + 右側「✏️」icon（點擊 → inline 展開編輯）
```

Inline textarea 規格：
- 高 88px，#FFFFFF 底，2px #06C755 border
- 底部：「儲存」（#06C755）+ 「取消」（ghost）兩個小按鈕

## 🟢 待確認（低優先，確認後再動）

### 6. 標籤刪除確認 Bottom Sheet 的數量提示

確認是否要顯示「此標籤已用於 N 張名片」。

- **顯示**（建議）：讓管理員知道影響範圍，避免誤刪
- **不顯示**：若 N 的數量難以即時計算可暫時省略

請 PM 確認後動工。

---

### 7. 標籤套用頁名片資訊字體

**問題**：標籤套用頁顯示的名片姓名字體偏小，不易快速確認是哪張名片。

**建議**：姓名從現有字體調大至 16px Bold。

請設計師確認視覺平衡後決定是否修改。
